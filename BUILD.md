# FileScout — Build Guide

This guide takes you from source code to a distributable
`FileScout-Setup.exe` (Windows) and `FileScout.dmg` (macOS).

---

## What You Need on YOUR Machine (the Developer)

| Tool | Version | Download |
|------|---------|----------|
| Node.js | 18 or higher | https://nodejs.org |
| Python | 3.9 or higher | https://python.org |
| Git (optional) | any | https://git-scm.com |

> Users who receive the final `.exe` or `.dmg` need **nothing** installed.

---

## Step 1 — Install Node dependencies

Open a terminal inside the `filescout/` folder and run:

```bash
npm install
```

---

## Step 2 — Install Python dependencies + PyInstaller

```bash
pip install -r python/requirements.txt
```

---

## Step 3 — Install Tesseract OCR (needed for PyInstaller bundle)

**macOS:**
```bash
brew install tesseract
```

**Windows:**
Download and install from:
https://github.com/UB-Mannheim/tesseract/wiki

After installing, make sure it is on your PATH:
```
tesseract --version
```

---

## Step 4 — Compile the Python script into a binary

This bundles Python + all pip packages + Tesseract bindings into one file.

```bash
# From the filescout/ folder:
npm run build:python
```

This creates:
- `python/dist/search.exe`  ← Windows
- `python/dist/search`      ← macOS / Linux

---

## Step 5 — Build the app installer

### Windows (.exe installer)

Run this on a **Windows machine** or a Windows VM:

```bash
npm run pack:win
```

Output: `dist/FileScout-Setup-1.0.0.exe`

### macOS (.dmg)

Run this on a **Mac**:

```bash
npm run pack:mac
```

Output: `dist/FileScout-1.0.0.dmg`

> To build both platforms you need to run each command on its respective OS.
> Cross-compilation (Mac → Windows) is not supported by electron-builder for .exe.

---

## Step 6 — Add App Icons (recommended before distributing)

Place your icon files at:

```
assets/
  icon.ico    ← Windows (256×256 minimum)
  icon.icns   ← macOS
  icon.png    ← 512×512 fallback
```

Free tools to convert a PNG to .ico / .icns:
- https://cloudconvert.com/png-to-ico
- https://cloudconvert.com/png-to-icns

---

## What the Installer Does for the User

### Windows
1. User double-clicks `FileScout-Setup.exe`
2. Installer wizard runs (they can choose install folder)
3. Desktop shortcut + Start Menu entry are created
4. **"Open with FileScout" is automatically added to the right-click menu** for folders — no extra steps needed

### macOS
1. User opens `FileScout.dmg`
2. Drags `FileScout.app` to Applications
3. Opens Terminal and runs **once**:
   ```bash
   bash /Applications/FileScout.app/Contents/Resources/scripts/mac-register-service.sh
   ```
4. Right-click any folder in Finder → Services → **Open with FileScout**

---

## Running in Development Mode

To run locally without building:

```bash
npm run dev
```

This starts React on port 3000 and Electron side by side.
The Python script is called directly (no binary needed in dev).

## Troubleshooting

**"Python not found" during build:python**
→ Make sure `python` or `python3` is on your PATH.

**Tesseract errors during search**
→ Re-install Tesseract and confirm `tesseract --version` works.

**Right-click menu not showing on Windows**
→ Restart Windows Explorer or sign out and back in.

**macOS service not appearing**
→ Run `mac-register-service.sh` again, then log out and back in.

**App not opening on macOS ("unidentified developer")**
→ Right-click the app → Open → Open anyway. Or run:
```bash
xattr -cr /Applications/FileScout.app
```
